
# 🐄 Sistema de Monitoramento de Produção Leiteira - FazenTECH

Este projeto tem como objetivo apresentar a **implementação de um esboço de sistema web** para o **monitoramento da produção de leite** em uma fazenda, com o intuito de otimizar o controle da produção e reduzir perdas. O sistema foi desenvolvido como parte de um **projeto acadêmico**, visando atender às necessidades de Lúcia, proprietária da fazenda.

  <img src="docs/preview.gif" alt="Tela inicial do sistema" width="70%">
</p>

## 📌 Objetivos

- Registrar a **quantidade diária de leite** produzida por cada vaca leiteira.
- Monitorar quem realizou o **manejo de cada vaca** e o respectivo horário da ordenha.
- Acompanhar o **tratamento de animais doentes**, registrando os dias necessários para a recuperação.
- Garantir que o sistema seja **simples, eficiente e com uma interface agradável**, atendendo às expectativas da Lúcia.

## 🛠️ Tecnologias Utilizadas

Este projeto foi desenvolvido utilizando as seguintes tecnologias:

1. **HTML5** - Para a estruturação da interface web.

## ▶️ Como Executar

### **1️⃣ Baixe os Arquivos**

Baixe todos os arquivos do projeto para o seu diretório local.

### **2️⃣ Execute o Código**

Abra o projeto no **Visual Studio Code** e clique com o botão direito sobre o arquivo **index.html**. Selecione a opção **"Open with Live Server"**. O navegador será aberto automaticamente, mostrando a interface do sistema de monitoramento.


```sh
open index.html
```

### **3️⃣ Interaja com o Sistema**

O sistema permitirá que você registre informações como:

- A quantidade de leite diária de cada vaca.
- O manejo realizado e o horário de cada ordenha.
- O status de tratamento dos animais doentes.

## 📌 Conclusão

Este projeto visa facilitar o monitoramento e registro da produção de leite na fazenda, contribuindo para a redução de perdas e aumento da eficiência no processo. A interface foi projetada para ser simples e otimizada para o uso no campo, ajudando a Lúcia a realizar o controle de sua produção de forma mais eficaz.
